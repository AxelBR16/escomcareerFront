🏆Nuestro Objetivo🏆


Desarrollar un sistema web y aplicación móvil que proporcione una orientación personalizada para la elección de carreras en la Escuela Superior de Cómputo (ESCOM), mediante una serie de pruebas interactivas, se evaluarán las aptitudes, preferencias y recursos financieros de cada aspirante para que, en base a los resultados, se ofrezcan recomendaciones de licenciaturas adecuadas.
